# B1 Funnel – Affiliate / Digital Product

This folder applies the global template to the AI Growth Toolkit stream. See `docs/b1_promo_sprint.md` for the current 48-hour promo plan.

- `copy.json` – overrides pulled into the landing page and HUD display.

Refer to `design_docs/businesses_plan.md` for positioning, KPIs, and target cadence.
